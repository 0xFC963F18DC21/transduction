TakeMTransducer

case class TakeMTransducer[A](n: Int) extends Transducer[A, A] with Product with Serializable

Mutable take transducer. Uses a mutable inner state to figure out how many items to accept. Do not reuse singular instances of this transducer!

A

Input type of reducer.

n

Number of items to take.
